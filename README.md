# Planora_2.0

**Система автоматизованого складання університетського розкладу**

##  Про проєкт
Цей застосунок розробляється в межах курсу "Програмна інженерія". Мета — автоматизувати складний процес формування розкладу, враховуючи обмеження аудиторій, викладачів та груп.
